# Apéndice — Guía de referencias

Este apéndice reúne las fuentes principales utilizadas en el proyecto en un formato pensado para el acceso del lector antes que para la cita académica. Los elementos están organizados por categorías: reportajes e investigaciones principales; fuentes analíticas, registrales y contextuales; fuentes de plataformas y canales; y elementos del archivo local.

Se aplican dos advertencias a lo largo de todo el apéndice.

Primera, la inclusión aquí no implica igual fiabilidad. El método del libro depende de la comparación, la atribución explícita y el manejo cuidadoso de fuentes con diferentes intereses y criterios probatorios. Una fuente que aparece en esta lista puede aparecer en el libro como un hallazgo principal, como un reportaje atribuido, como comentario de un Estado hostil o como propaganda que se analiza antes que se acepta. Los lectores deben seguir las atribuciones en el texto y las notas al pie de cada capítulo para entender cómo se utilizó cada fuente.

Segunda, algunos materiales se preservaron como archivos locales, capturas de plataformas o espejos antes de que pudieran confirmarse las URLs canónicas de publicación. Donde no había disponible una URL canónica limpia, la fuente aparece como archivo local. Estos elementos siguen siendo citables desde el archivo del proyecto y deben normalizarse a URLs completas antes de cualquier publicación impresa definitiva.
