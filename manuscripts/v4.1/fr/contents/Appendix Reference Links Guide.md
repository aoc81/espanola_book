# Annexe — Guide des liens de référence

Cette annexe rassemble les principales sources utilisées dans le projet, dans un format pensé pour l'accès du lecteur plutôt que pour la citation académique. Les éléments sont organisés par catégorie : grands reportages et enquêtes ; sources analytiques, de registre et de contexte ; sources issues des plateformes et canaux ; et éléments d'archives locales.

Deux réserves valent pour l'ensemble.

Premièrement, la présence d'une source ici n'implique pas qu'elle présente le même degré de fiabilité que les autres. La méthode du livre repose sur la comparaison, l'attribution explicite et le maniement prudent de sources ayant des intérêts et des standards probatoires différents. Une source figurant dans cette liste peut apparaître dans le livre comme constat principal, comme reportage attribué, comme commentaire d'un État hostile, ou comme propagande analysée plutôt qu'acceptée. Les lecteurs doivent suivre les attributions dans le texte et les notes de fin de chapitre pour comprendre la manière dont chaque source a été utilisée.

Deuxièmement, certains matériaux ont été conservés sous forme d'archives locales, de captures de plateformes ou de miroirs avant qu'une URL de publication stable puisse être confirmée. Lorsqu'aucune URL canonique propre n'était disponible, la source est indiquée comme archive locale. Ces éléments restent citables depuis les archives du projet et devront être normalisés en URL complètes avant toute publication imprimée définitive.
