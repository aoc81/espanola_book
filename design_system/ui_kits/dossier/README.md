# Dossier Kit — Intelligence Case File browser

A dark-mode "declassified file folder" interface, complementary to the
article kit. Used for chapter openers, the "Subjects" and "Map" surfaces
of the investigation, and standalone evidence pages.

## Components

- `DossierShell.jsx` — full-page dark frame with grain + corner registration marks
- `DossierHeader.jsx` — case-file masthead (file id, status, last-update)
- `SubjectGrid.jsx` + `SubjectCard.jsx` — grid of redacted subject cards
- `EvidenceList.jsx` — typewritten list of evidence items
- `MapPanel.jsx` — coordinate-stamped map placeholder
- `Sidebar.jsx` — left navigation (chapters + filters)
