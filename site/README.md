# Website

Vite + React web reader for the canonical manuscript tree.

## Commands

- `npm install`
- `npm run dev`
- `npm run build`
- `npm run preview`

## Content source

The site content is generated by `../scripts/website/build_site_content.mjs`.

Default input edition:

- `../manuscripts/v4.1/en/contents`
- `../assets/manifests/v4.1/en/chapter_images.json`
- `../assets/images/manuscripts/v4.1/en`

The generator writes to:

- `../build/site/generated-manuscript.js`
- `../build/site/public/pictures`

The site also serves its public design assets from `../build/site/public`, which the generator repopulates from `../design_system/`.
